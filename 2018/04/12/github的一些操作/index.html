github的一些操作

1.本地建立git仓库，先将窗口切换到本地项目下

git init

2.将项目的所有文件添加到仓库中

git add .

3.将add了的文件添加到本地仓库

git commit -m  ‘测试语句’

4.到github上创建自己的远程仓库repository,拿到创建仓库的https地址 5.将自己本地仓库关联到远程仓库中

git remote add origin  https://github.com/yf/Test1

6.传之前先pull下master分支

git pull  origin master

7.上传本地代码

git push -u  origin master
